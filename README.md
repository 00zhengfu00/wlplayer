# wlplayer
### Android 基于：FFmpeg+OpenSL+OpenGL+Mediacodec 的视频播放SDK，可播放网络、本地和广播等流媒。支持当前播放页直接切换播放源；支持视频实时截图；优先选用GPU解码，解码速度更快；在手机支持1080P、2K、4K等的情况下都可播放；封装常用播放状态回调，接入简单。
## APP Demo（注：视频数据来自于熊猫TV，广播数据来自于中国广播网）
