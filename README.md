# wlplayer
### Android 基于：FFmpeg+OpenSL+OpenGL+Mediacodec 的视频播放SDK，可播放网络、本地和广播等流媒体，有限选用硬件解码器解码，在手机支持1080P、2K、4K等的情况下都可播放。
## APP Demo（注：视频数据来自于熊猫TV，广播数据来自于中国广播网）
